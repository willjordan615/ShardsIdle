Outstanding / Notes

Belt display for reusable items — lockpick, rope, and quest items show a quantity
  that never decrements. Needs a dedicated UI pass to either suppress quantity display
  or show a different indicator (carried/equipped vs count) for non-consumable belt items.
  Quest items have the same problem throughout.

Child proc consumable decrement — confirmed working. Decrements by matching
  parentSkillId against item skillID/effect_skillid. Edge case: child with two consumable
  parents could double-decrement on a single proc. Acceptable for now.

Teleport escape timing — the escape flag is set on button press but the full combat result
  must arrive before the escape POST fires. Combat runs to completion server-side regardless.
  This is by design — the result is needed to know the combatID.

Tier intentional exceptions — greatsword_verdict_tier5 (Hall of Ancestral Echoes, diff 7)
  and greataxe_rage_tier5 (Stonetusk, diff 7) are above the normal drop tier for their
  difficulty. This is intentional — hardwired named loot is exempt from the global tier range.

Smoketest recommended — escape consumable flow end to end, cleanse fix on purge/elixir,
  flat heals still firing correctly after all the skills.json edits this session.
