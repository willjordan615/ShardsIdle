Feature — Belt Modal + Stack-Aware Consumables (inventory-system.js)

Feature — Loop Toggle Button (combat-log.js, index.html, styles.css)
Stop button hidden (not removed). Loop toggle replaces it — green, always visible. Active (lit) when looping, dimmed (opacity 0.4) while pending exit, off when stopped. Toggling off calls requestLoopExit for graceful stop. Toggling on calls startCombat directly. startCombat no longer clears pendingLoopExit — _fireNextCombat owns that transition.
Fix — Secret Path Threshold Guarantee (combatEngine.js)
All four check types (skill, stat, combo, item_and_stat): meeting or exceeding difficultyThreshold now guarantees success (1.0). Under-threshold attempts retain the probabilistic roll, floored at 5%, ceiling removed.
Fix — Combo Hint Party Check (game-data.js)
getComboHints now includes skills from all currentState.currentParty members in knownIDs, not just the viewed character.
Feature — Discovery Origin Line (character-management.js)
Discovery progress bar cards now show ⚗ Discovered via: Parent One + Parent Two below the description. Sourced from skillDef.parentSkills. Skills without two parents show nothing.
Fix — AI Damage Skill Jitter (combatEngine.js)
Damage skills now get a 50%–150% random multiplier on base score. Prevents lock-in when two damage skills are equipped — neither can ever be crowded to 0% chance.
Admin Panel — Character View (admin-panel.js)
adminViewCharacter now shows full skill list in DB order with level and equipped position, plus equipment. Used to diagnose Marcel's secret path issue.
