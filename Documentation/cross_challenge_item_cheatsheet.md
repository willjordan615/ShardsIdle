## Authoring Notes

- `ferryfolk_lantern` and `coronation_writ` are currently dead ends — no challenge uses them yet. Wire them in or note them as future hooks.
- `spire_fragment` is stackable; all others are not.
- `verdant_word_seal` has the broadest network — appears in 19 challenges. Safe to reference widely; players will accumulate it naturally.
- `oath_stone` feeds both the Elara arc (gallery → reconciliation) and the Orc alliance arc (blood_oath_tribunal). Central item for Act 2–3.
- When creating new challenges in d9–d15, check this table before inventing new cross-challenge items — most situations are already covered by existing ones.
